EDS Contacts Integration

Version 2.0.1

Helper local requis

L’extension ne peut pas accéder seule à Evolution Data Server. Installez d’abord EDS Contacts Helper.

Afficher les commandes d’installation 
sudo apt install python3 python3-gi \
  gir1.2-edataserver-1.2 \
  gir1.2-ebook-1.2 \
  gir1.2-ebookcontacts-1.2

unzip eds-contacts-helper-*.zip
cd eds-contacts-helper
./install-native-helper.sh

Consentement et synchronisation

Les modifications et suppressions effectuées dans Thunderbird ne sont pas encore propagées vers EDS.

Diagnostic

Le test lit les contacts EDS uniquement après votre consentement.
