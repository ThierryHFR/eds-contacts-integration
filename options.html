EDS Contacts Integration

Téléchargement du helper :

https://github.com/ThierryHFR/eds-contacts-helper/releases

Installation

		sudo apt install \
		  python3 \
		  python3-gi \
		  gir1.2-edataserver-1.2 \
		  gir1.2-ebook-1.2 \
		  gir1.2-ebookcontacts-1.2

		unzip eds-contacts-helper-*.zip
		cd eds-contacts-helper
		./install-native-helper.sh

Utilise ce bouton pour vérifier que le helper Native Messaging est installé et peut lire Evolution Data Server.
